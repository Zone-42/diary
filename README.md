# Diary

This repository stores personal diary entries and publishes them using GitHub Pages.

## Structure

- `index.html` - landing page for the diary site
- `assets/styles.css` - styles for the website
- `entries/` - markdown diary entries

## Add a new entry

1. Create a new file in `entries/` using a date name, for example: `2026-02-13.md`
2. Add a link to that file in `index.html`
3. Commit and push

## Publish on GitHub Pages

1. Create a GitHub repository and push this project
2. In GitHub, open **Settings -> Pages**
3. Set **Source** to **Deploy from a branch**
4. Choose branch `main` and folder `/ (root)`
5. Save and wait for GitHub to publish

The site will be available at:

`https://<your-username>.github.io/<repository-name>/`
